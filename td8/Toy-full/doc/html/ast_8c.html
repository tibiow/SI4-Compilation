Detailed Description

The module ast is in charge of allocating and freeing all the elements of the Abstact Syntax Tree (AST). For each element xxx which can be stored in the AST, we need to declare two functions:

  • ast_node *make_xxx(....) in charge of the new node allocation. Parameters depend of the node which need to be created (e.g. and if-then-else node takes 3 ast as parameters, whereas an integer constants takes just one int
  • void free_xxx(ast_node *node) in charge of freeing the given node
